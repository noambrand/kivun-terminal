========================================
MAINTENANCE GUIDELINES
========================================

When adding new files:
- User utilities → Root directory
- Diagnostic tools → DEVELOPMENT/
- Bundled installers → BUILD_FILES/
- Backups → Create new BACKUP_vX.X.X/ folder

Before major changes:
1. Create new backup folder (BACKUP_vX.X.X/)
2. Copy all working files
3. Document in BACKUP_README.txt

When updating version:
1. Update CHANGELOG.md
2. Create RELEASE_NOTES_vX.X.X.md
3. Rebuild installer (Kivun_Terminal_Setup.exe)
4. Create backup of working state
