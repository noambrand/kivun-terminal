================================================================================
  CLAUDEHEBREW - DEFAULT CREDENTIALS
================================================================================

For installations using: install-wsl-ubuntu-simple.bat

Default credentials (from config.txt):
Username: username
Password: password

To customize: Edit config.txt BEFORE running the installer.

================================================================================

IMPORTANT - PASSWORD TYPING IN UBUNTU:

When Ubuntu asks for a password, NOTHING APPEARS ON SCREEN while you type.
No dots, no asterisks, no characters - NOTHING.

This is NORMAL Linux security behavior.

Just type: password
Then press: Enter

Even though you can't see what you're typing, it's working!

================================================================================

WHAT THESE ARE USED FOR:

1. Logging into Ubuntu
2. Running "sudo" commands (installing packages, system administration)
3. The post-install.sh script needs sudo to install:
   - Konsole terminal
   - Hebrew fonts
   - Node.js
   - Claude Code CLI

================================================================================

CUSTOMIZATION:

To use different credentials:
1. Open config.txt in a text editor
2. Change USERNAME and PASSWORD values
3. Save the file
4. Run install-wsl-ubuntu-simple.bat

The installer will use your custom credentials.

================================================================================

SECURITY:

These generic credentials are fine because:
- Ubuntu runs isolated in WSL on your local machine
- Not accessible from outside your computer
- Used only for package installation

For detailed security information, see: SECURITY.txt

If you want to change the password AFTER installation:
  In Ubuntu, type: passwd
  Follow the prompts
