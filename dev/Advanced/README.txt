# Advanced Installation Options

These installers are for specific use cases. Most users should use:
  install-wsl-ubuntu-simple.bat (in parent directory)

## When to use these:

### install-claude-hebrew-improved.bat
- You want to choose your own username/password during Ubuntu first launch
- You prefer manual control over credentials

### auto-setup-ubuntu.bat
- You want Ubuntu username derived from Windows username
- Auto-generated password is acceptable (Ubuntu2024!)

### install-wsl-ubuntu-auto.bat
- Similar to auto-setup-ubuntu.bat
- Alternative automated installation approach

## Recommended Path
For most users, use the simple installer:
1. Edit config.txt (optional)
2. Run install-wsl-ubuntu-simple.bat as Administrator
3. Follow instructions in START_HERE.txt
